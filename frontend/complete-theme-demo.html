🚀 How to Use

For Users:

  • • Click the sun/moon icon in the header to toggle themes
  • • Theme preference is automatically saved
  • • Setting persists between browser sessions
  • • Smooth transitions provide visual feedback

For Developers:

  • • Import and use useTheme() hook
  • • Use useThemeClasses() for consistent styling
  • • Wrap app with <ThemeProvider>
  • • Aviation colors work in both themes
